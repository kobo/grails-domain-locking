Domain Locking Plugin for Grails 3
==================================

This is a Grails 3 plugin to provide an easy way to use optimistic/pessimistic locking.


Installation
------------

In build.gradle:

    dependencies {
        compile "org.grails.plugins:domain-locking:<VERSION>"
    }
